# douban-movie

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```

最后打开浏览器，输入`localhost:8880`即可访问。效果图如下：
